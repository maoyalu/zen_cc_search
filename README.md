## How to install

**Assumption**: Users will be familiar with CLI.

1. Clone and go into the root directory

```bash
git clone https://github.com/maoyalu/zen_cc_search.git

cd zen_cc_search
```

2. Restore project requirements

```bash
pip install -r requirements.txt
```

3. Run

```bash
python -m zensearch
```
